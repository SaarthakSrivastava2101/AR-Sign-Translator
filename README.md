\# AR Sign Translator

\## Features

\- Real-time sign language recognition

\- Python-based ML processing

\- C# / .NET application integration

\- Dataset generation and live capture



\## Tech Stack

\- Python (OpenCV, ML)

\- C# (.NET / Unity compatible)

\- Custom datasets (WLASL-style)



\## Project Structure

\- AI / Core / Utils: C# logic

\- Python scripts: data processing \& live capture

\- Dataset / Output: training and results
